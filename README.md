# CreateJS Canvas Editor

##  Project Description
This is an **ES6 class-based CreateJS Canvas Editor**. It allows users to drag-and-drop three predefined items — **Heart**, **Grid‑Square**, and **"Hello World” text** — and interact with them easily.

### Features
- **Select** – Click an object to select it (adds a blue border)
- **Drag & Drop** – Move items freely on the canvas
- **Color Palette** – Change Heart & Text colors (Red, Purple, Green)
- **Grid Cell Toggle** – Click grid cells to toggle between red and transparent
- **Clone Items** – Press **Ctrl + C** to duplicate selected items
- **Theme Toggle** – Click the toggle shape to switch between Light & Dark mode

## How to Run the Project
**Step1: Clone the repository**
```bash
git clone https://github.com/Sabra-jain/CanvasEditor.git
```

**Step2: Open the project folder**
```bash
cd createjs-canvas-editor
```

**Step3: Simply open index.html in any modern browser (e.g. Chrome).**
